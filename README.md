# 📈 CodeAlpha Stock Portfolio Tracker - Task 2

## 🎯 Project Overview
This is Task 2 for the CodeAlpha Python Programming Internship. A command-line Stock Portfolio Tracker that helps users manage investments using predefined stock prices, calculate total portfolio value, track profit/loss, and export data to CSV.

## ✨ Features
- **View Stock Prices**: Display 5 available stocks - AAPL, TSLA, GOOGL, MSFT, AMZN
- **Add Stocks**: Add any stock symbol with quantity to your portfolio
- **Remove Stocks**: Delete stocks from portfolio by symbol
- **View Portfolio**: Display holdings in clean table format with total value
- **Calculate Profit/Loss**: Compare current value with your invested amount
- **Save Portfolio**: Export Stock and Quantity to `portfolio.csv`
- **Load Portfolio**: Import previously saved portfolio from CSV
- **Error Handling**: Handles invalid stocks, negative quantity, empty portfolio
